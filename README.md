# GrindCo

My team and I worked on an end-to-end cloud migration project for a café concept called GrindCo. I was on the cloud infrastructure track, where I was responsible for setting up our serverless backend pipeline using AWS Lambda and configuring our Amazon DynamoDB database schemas to handle real-time orders and table bookings cleanly. My other teammates focused on building the interactive front-end interface. The project is not currently live at http://grindco-web-portal-gwoup2.s3-website.eu-north-1.amazonaws.com/ because we tore down the resources post-testing, but during development, we used Amazon S3 static website hosting to make it live on the web. We also integrated AWS Cognito User Pools to securely handle customer account creation and sign-in verification
